# ALIA | Grupo SINAI  
***Documentación | Datos Discriminativos: Discurso de Odio (Hate Speech)***

Este subdirectorio contiene la documentación asociada a los procesos de recolección, curación y anotación de datos para tareas discriminativas sobre discurso de odio, adaptada al repositorio de documentación de ALIA.

Fomentamos la apertura y reproducibilidad en nuestras investigaciones; por ello documentamos los pasos, scripts y artefactos relevantes que acompañan los pipelines de datos.
